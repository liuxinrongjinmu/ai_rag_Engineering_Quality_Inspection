# ALIGNMENT - 工程质检RAG系统

## 一、原始需求

构建一个公路工程质量检测RAG（检索增强生成）系统，支持：
1. **本地知识库检索**：基于11个PDF规范文档和3个Excel数据文件
2. **网络检索补充**：当本地知识库无答案时，联网搜索权威来源
3. **答案可追溯**：每个答案必须能追溯到具体的规范条文、页码、章节

## 二、边界确认

### 2.1 任务范围

| 包含 | 不包含 |
|------|--------|
| 后端API服务 | 前端界面 |
| PDF/Excel解析与入库 | 用户管理系统 |
| 向量数据库构建 | 多租户支持 |
| 本地检索+网络检索混合 | 实时协作功能 |
| RAG问答接口 | 移动端适配 |
| 来源追溯功能 | 离线部署方案 |

### 2.2 数据范围

**PDF文档（11个）：**
- JTG F80-1-2017 公路工程质量检验评定标准（核心标准）
- JTG 3420-2020 水泥及水泥混凝土试验规程
- JTG 3431-2024 岩石试验规程
- JTG 3432-2024 集料试验规程
- JTG 3441-2024 无机结合料稳定材料试验规程
- 公路土工试验规程2020
- DB43T1206-2016 公路工程监理规范
- JT/T1181-2018 试验检测等级管理要求
- 常用45项公路试验检测项目、频率及取样要求
- P020240607597845867899.pdf（待确认内容）

**Excel文件（3个）：**
- 公路工程试验常规检测项目、检测标准、检测频率、取样方法（55行）
- 公路工程单位分部分项工程划分表（206行）
- 检测频率_extract（315行，最详细）

### 2.3 用户画像

| 用户类型 | 核心需求 | 提问特点 |
|---------|---------|---------|
| 甲方监理 | 合规性验证、责任界定 | 关注标准依据、检测频率 |
| 总工程师 | 规范依据、条文解释 | 专业术语、跨文档关联 |
| 现场质检员 | 快速答案、操作指引 | 口语化、实际问题导向 |
| 企事业单位 | 标准查询、报告编制 | 系统化、完整信息 |

## 三、需求理解

### 3.1 核心功能流程

```
用户提问 → 意图理解 → 本地检索 → 有答案? 
                                    ↓
                        是 → 生成答案+来源追溯 → 返回
                        否 → 网络检索 → 过滤权威来源 → 生成答案+标注来源 → 返回
```

### 3.2 关键技术决策

| 决策点 | 选择 | 理由 |
|-------|------|------|
| PDF解析 | PyMuPDF | 支持文本+表格提取，性能优秀 |
| 向量数据库 | ChromaDB | 轻量级、易部署、支持本地持久化 |
| Embedding | text-embedding-3-small / bge-large-zh | 中文效果好、性价比高 |
| LLM | GPT-4o / DeepSeek | 中文专业术语理解能力强 |
| 网络检索 | Tavily API / Serper | 专业搜索API，支持权威来源过滤 |
| 后端框架 | FastAPI | 异步支持好、文档自动生成 |

### 3.3 切片策略

**PDF文档切片：**
- 按章节/条文切分（优先）
- 保留文档结构信息（标题层级）
- 切片大小：500-1000字符
- 重叠：100字符

**Excel数据切片：**
- 按行记录切分
- 保留列名作为上下文
- 构建结构化描述文本

## 四、三条底线

### 4.1 权威性底线
- 网络检索结果**永远**标注来源
- 本地知识库答案优先级高于网络检索
- 网络检索结果必须标注"非官方来源，仅供参考"

### 4.2 可解释性底线
- 每个答案必须追溯到具体规范条文、页码、章节
- 用户可查看原始文档片段
- 不确定时明确告知，不装作确定

### 4.3 质量底线
- 数据清洗完整，不假装成功
- 切片策略合理，不支离破碎
- 检索准确率≥80%
- 代码可维护，有注释

## 五、验收标准

### 5.1 测试场景（5个）

| 场景 | 问题 | 预期结果 |
|------|------|---------|
| 基础规范查询 | "土方路基压实度检测频率是多少？" | 返回JTG F80-1-2017具体要求+条文号 |
| 跨文档关联 | "水泥混凝土抗压强度试件取样方法和检测频率？" | 同时引用JTG 3420-2020和检测频率表 |

| 来源验证 | 点击"查看原文" | 跳转到原始PDF位置或显示原文片段 |
