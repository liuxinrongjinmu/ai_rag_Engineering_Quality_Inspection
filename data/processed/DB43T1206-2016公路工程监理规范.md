# 3 术语和定义

下列术语和定义适用于本规范。 

# 3.1 监理

监理机构及监理人员对工程质量、环保、费用、进度等实施监督管理和咨询服 务,并履行工程安全生产管理法定职责的活动。 

# 3.2 监理企业

具有法人资格并取得相应的公路工程监理资质证书的企业。 

# 3.3 建设单位

也称为项目业主,指工程项目的法定投资主体,是对工程项目的策划、建设(也 可能包括建成后的生产经营)实行 条龙管理和全面负责的主体。 

# 3.4 监理机构

履行监理职责的现场监理组织。 

# 3.5 总监理工程师办公室

简称“总监办”。 工程现场设立一级监理机构时,即为监理机构;设立二级监理 机构时,为项目驻地监理工程师办公室的上级监理机构。 

# 3.6 驻地监理工程师办公室

简称“驻地办”,是总监办下设的二级监理机 构。 

# 3.7 监理工程师

具有公路工程监理工程师或专业监理工程师资格、从事监理工作的人员。 

# 3.8 总监理工程师

简称“总监”,具有公路工程监理工程师资格,主持监理机构工作,全面负责履 行监理职责的管理者。 

# 3.9 驻地监理工程师

具有公路工程监理工程师资格,经总监理工程师授权的驻地监理机构负责人。 

# 3.10 专业监理工程师

具有公路工程监理工程师或专业监理工程师资格,经总监理工程师或驻地监 理工程师授权 负责实施某一专业或某一岗位监理工作的监理工程师 

# 3.11 试验检测监理工程师

通过相应的试验检测工程师执业资格认定 经总监理工程师或驻地监理工程 师授权 负责试验检测监理工作的监理工程师 

# 3.12 计量监理工程师

具有工程造价相应执业资格 经总监理工程师或驻地监理工程师授权 负责计 量监理工作的监理工程师。 

# 3.13 监理员

通过相应的监理业务水平认定、从事具体监理工作的人员。 

# 3.14 试验员

通过相应的试验检测业务水平认定、从事试验检测工作的人员。 

# 3.15 监理人员

从事监理工作的专业技术人员的统称。 

# 3.16 监理计划

由总监理工程师主持编制、全面开展监理工作的指导性文件。 

# 3.17 监理细则

根据监理计划,针对专业性较强、危险性较大的分部分项工程或监理工作的某 一方面 由驻地监理工程师主持编写 经总监理工程师批准的操作性文件 

# 3.18 巡视

监理人员对施工现场定期或不定期进行的日常巡视检查活动。 

# 3.19 旁站

监理人员在施工现场对某一具体的工序、工艺或部位施工全过程进行的现场 监督活动。 

# 3.20 抽检

监理机构按规定的项目和频率 见附录 对工程材料或工程实体质量进行的 平行或随机检验活动。 

# 3.21 见证

监理人员对施工单位进行的涉及结构安全的试块、试件及主要工程材料现场 取样、封样、送检的工作以及关键项目现场检测过程的监督活动。 

# 3.22 试验工程

为确认施工方案或获取控制参数所进行的试验性路段或工程部位。 

# 3.23 标准试验

为确定工程材料的最佳组合(如级配、配合比等)或建立施工控制和检验标准 所进行的试验 

# 3.24 缺陷责任期

施工单位按照合同约定承担缺陷修复义务的期限 

# 3.25 工程延期

由于非施工单位原因造成合同工期延长。 

# 3.26 工程延误

由于施工单位自身原因造成施工期延长。 

# 3.27 监理日志

监理机构每日对日常监理工作开展情况所做的记录。 

# 3.28 巡视记录

监理工程师对施工现场和施工的主要工程巡视检查(见附录 )所做的记录。 

# 3.29 旁站记录

监理人员对试验工程、关键部位(见附录 )的施工过程进行旁站所做的记录。 

# 3.30 监理月报

监理机构每月向建设单位和监理企业提交的项目工程监理工作及工程实施情 况等分析、总结报告。 

# 3.31 公路机电工程监理

公路监控、通信、收费、供配电、照明、隧道机电系统等工程监理的统称。 

# 3.32 自管模式

由项目建设单位按照 公路工程施工监理规范 和本规范要 求设置现场监理机构,配备具有相应执业资格的专业人员,履行相应监理职责,将 监理工作纳入其建设管理的一种项目管理组织形式。 

# 基本规定

4.1 公路工程监理应依据下列法律、法规、文件开展工作: 

国家和地方法律、法规。 

国家、行业、地方有关标准、规范、规程。 

监理、施工合同文件。 

工程设计文件和图纸。 

其他有关工程管理文件 

4.2 公路工程监理企业应公平、诚信、科学、自律地开展监理与相关服务 活动。 

4.3 公路工程监理,除应符合本规范外,还应符合国家和行业现行有关标准、 规范的规定。 

4.4 监理合同中必须明确建设单位与监理企业的职责与权限,应避免责权不 清和交叉。 

4.5 建设单位应严格执行公路工程质量管理、安全生产、环境保护等法律法 规 提供合法 规范 有序的监理工作环境 

4.6 公路工程施工监理分为下列三个阶段: 

1 施工准备阶段:监理合同签订之日至工程开工令确定的开工之日。 

施工阶段:工程开工之日至工程交工验收申请受理之日。 

3 验收与缺陷责任期阶段:工程交工验收申请受理之日至缺陷责任终止证书 签发之日。 

4.7 监理服务费由合同双方依据监理机构的设置、人员、设备与设施配备、服 务时间 服务内容 服务成本 服务质量和市场供求状况等因素确定 并符合国家法 律法规的有关规定。 

4.8 公路工程项目监理工作实行总监理工程师负责制,总监理工程师应在岗 履职。 

4.9 监理企业应选派具备相应资格的监理人员进驻项目现场,总监理工程师 应组织项目监理人员采取巡视、旁站和平行检验等形式实施工程监理,按照规定对 施工单位报审的建筑材料、建筑构配件和设备进行查验,不得将不合格的建筑材 料、建筑构配件和设备按合格签字。 

4.10 总监理工程师发现施工单位未按照设计文件施工 违反工程建设强制 性标准施工或者发生质量事故的 应当及时签发工程停工令 

4.11 监理机构在监理过程中发现施工存在质量问题或安全事故隐患时,应 签发监理工作指令,要求施工单位整改;未整改或整改不符合要求的,施工单位不 得进入下一道工序施工 不得进行计量与支付 情况严重时 应签发工程停工令 施工单位拒不整改或停工的 监理机构应及时向建设单位和有关部门报告 

4.12 在监理实施过程中,应逐步推广采用计算机、网络和通信等现代信息 技术。 

# 5 监理机构设置及职责

# 5.1 监理机构设置

511 公路工程实施监理时,应组建监理机构。监理机构的组织形式和规模, 可根据公路工程的特点、规模、技术复杂程度、环境等因素确定。 

512 高速公路和一级公路可设置两级监理机构 即总监办和驻地办 当工 程规模较小时宜设置一级监理机构,即总监办。当驻地办承监范围内有多个施工 承包单位或多个独立工区时,还可设置驻地监理组。 

513 同一建设单位 有多个规模较小的公路工程建设项目 计划同时或相继 开工时可合并设置一级监理机构,即总监办。总监办可按项目或监理职能设置监 理组 并合理调配人员 

514 监理机构一般应建立工地试验室。 

# 5.2 监理人员配备

521 监理机构中监理人员应由总监理工程师 监理工程师 试验检测人员和 必要的监理员等组成 监理人员的数量和专业结构 应根据监理内容 工程规模 合同工期和施工阶段等因素,按保证有效监理的原则确定。 

522 高速公路 一级公路等宜按每年每 万元建安费配备监理工程师 名。根据工程特点和实际需要,上述配置可在 $0 . 8 \sim 1 . 2$ 的系数范围内调整。其他 等级公路,根据实际调整配备。 

523 总监办应配备 名总监理工程师和若干名专业监理工程师 总监理工 程师一般应具有相关主管部门核准的公路工程监理执业资格和相应的专业技术职 称 五年以上的现场工程监理经历 担任过一项类似工程的总监理工程师或副总监 

理工程师或者两项类似工程的驻地监理工程师 总监办可设置副总监理工程师 其任职条件同驻地监理工程师 

524 驻地办应根据工程复杂程度配备 名驻地监理工程师和若干名专业监 理工程师。驻地监理工程师一般应具有相关主管部门核准的公路工程监理执业资 格和相应的专业技术职称 类似工程三年以上的监理经历 驻地办可设置副驻地 工程师 其任职条件同专业监理工程师 

525 当工程规模较小或技术复杂程度较低时 监理工程师的任职条件 人员 配备数量可适当降低 

526 专业监理工程师及以上人员的变更应经建设单位同意 并报质量安全 监督管理机构备案。

7112 监理工程师应对完工后无法检验的隐蔽工程进行检查验收 签认 留 存影像资料。验收不合格应拒绝签认,同时应要求施工单位在指定的时间内整改 并重新报验 未经签认不得进行下道工序施工 

7113 监理机构收到施工单位中间交工申请后,应检查施工单位工序检验记 录及隐蔽工程签认记录,检查交验工程的质量保证资料的完整性和检验评定资料 的规范准确性,根据《公路工程质量检验评定标准》( )进行工程质量检验 评定,对合格的分项工程签发中间交工证书。 

7114 驻地办应及时对已完分部工程进行质量检验评定, 总监办应及时组织 对单位工程和合同段 程进行质量检验评定。 

7115 监理机构在监理过程中发现施工质量不符合法律法规、工程建设强制 性标准、工程设计文件和施工合同等规定的,应建立质量问题处理台账,并按下列 规定处理 

1 对质量不合格的工程材料、构配件和设备,监理人员不得签认,并应签发监 理工作指令,要求施工单位限期清出施工现场。 

对存在质量缺陷或质量不合格的工程,监理机构应签发监理工作指令,要 求施工单位整改或返工。施工单位返工或整改结束并自检合格后,监理工程师应 对返工或整改结果进行复查并重新验收 

对需要加固补强的工程 监理机构应签发监理工作指令 要求施工单位报 送经建设 设计等相关单位认可的处理方案 对处理过程跟踪检查 并对处理结果 重新验收。 

对可能危及结构安全或存在重大隐患的质量问题 应签发停工令 并向建 设单位报告。 

7116 对工程质量事故,监理机构应依法依规按程序报告处理。 

# 7.2 安全监理

721 监理机构应根据法律法规 工程建设强制性标准履行工程建设安全生 产管理的监理职责。根据监理合同配置安全监理人员,编制安全监理计划及安全 监理细则 明确监理人员的安全监理职责 

722 监理机构应检查施工单位的安全生产规章制度及其执行情况,检查施 工单位安全生产许可证 检查施工单位企业负责人 项目负责人 专职安全员和特 种作业人员的资格 检查特种设备的安全许可验收手续 

723 工程开工前 监理机构应审查施工组织设计中的安全技术措施或专项 施工方案是否符合工程建设强制性标准和设计要求 应同时审查应急预案 平安工 地建设方案和重点工程施工安全风险评估报告,经审查符合规定方可批准开工。 

724 超过一定规模的危险性较大的分项工程专项施工方案 应检查施工单 位组织专家论证评审的情况以及附具的安全验算结果 

725 分项 分部工程开工前 应检查施工单位的施工安全技术交底 并做好 检查记录。 

726 监理机构应检查施工单位安全生产教育和培训计划 督促施工单位对 从业人员进行安全生产教育和培训 并做好检查记录 

727 监理机构应建立施工安全重大危险源管理制度,加强对工程项目施工 安全重大危险源的动态管理 建立工程项目施工安全重大危险源监理台账 

728 监理机构应检查生产、办公、生活现场安全防护措施及安全警示标志是 否健全 检查施工单位主要施工机械 设备 临时用电等的检测检验报告和验收手 续,巡视检查危险性较大的分项、分部工程专项施工方案实施情况。 

729 监理工程师在监理实施过程中发现施工单位未按有关安全法律、法规、 工程建设强制性标准 设计要求 已批准的方案施工 或存在安全事故隐患时 应立 即发出书面指令要求施工单位整改 情况严重时 应签发工程停工令 要求施工单 位暂停施工,并同时报告建设单位。若施工单位拒不整改或不停止施工,应及时向 

建设单位和有关主管部门报告。 

7210 监理机构应根据国家的有关法律法规的规定 对安全生产专项费用的 计量与支付按照施工单位申报、监理机构审查、建设单位审批支付的程序实施。监 督施工单位安全生产专项费用专款专用。 

7211 监理机构应积极参与或协助安全事故调查与处理,及时向建设单位提 交事故书面报告,并应将完整的处理记录整理归档。 

7212 分项工程验收时,安全事故的现场未处理完成,或安全隐患未消除,不 得签发中间交工证书。 

7213 监理机构应由专人负责建立施工安全监理台账,收集、整理相关安全 监理资料。 

# 7.3 环保监理

731 监理机构应根据法律、法规、合同文件,履行工程建设施工环境保护的 监理职责,将施工环境保护监理工作的内容纳入监理计划及监理实施细则。 

732 监理机构应按设计文件和环境影响评估报告的要求,审查施工组织设 计 分部分项工程开工申请报告 合格后方可批准开工 

733 监理机构在工程建设施工过程中应检查环境保护措施的落实情况。检 查的主要内容: 

施工单位是否落实了施工环境保护责任人。 

施工现场的布设是否符合环保要求。 

职业危害的防护措施是否健全。 

降尘、降噪,防治污染是否符合相关要求。 

取、弃土是否在批准的场地内进行,取、弃土结束后是否采取了有效的排 水、防护和植被恢复措施。 

734 发现未按合同要求落实环境保护措施、违反环境保护有关规定,应签发 监理工作指令要求施工单位整改 情况严重的 应签发工程停工令要求施工单位暂 停施工,并及时报告建设单位。 

735 发现文物时,监理机构应签发监理工作指令要求施工单位依法保护现 场,并及时报告建设单位。 

736 监理机构应要求施工单位按照砍伐许可证批准的砍伐面积、树种、株数 进行砍伐,应督促施工单位依法保护野生动物、植物、水域和自然景观。 

# 7.4 费用监理

741 监理机构应按照计量规则、计量程序和合同文件的相关约定进行工程计 量 做到客观 公正 准确 及时 未经总监理工程师签认 建设单位不得支付工程款 

742 已完工程计量与支付的先决条件是其质量经过施工单位自检和监理机 构验收确认合格,且质量保证资料真实、齐全、有效,同时应符合安全和环保监理的 规定。 

743 监理机构收到施工单位计量申请后,应以建设单位核准的工程量清单 为依据及时计量,对于复杂、有争议、需要现场确认的项目,应会同建设、设计、施工 等单位现场计量 

744 施工单位提交的支付申请应附有相关支持性文件资料,监理工程师应 在合同规定的时间内审核并签发支付证书。 

745 监理机构依据签发的中间交工证书及核定的工程量等资料进行工程计 量,审查计量支付台账,对计量支付台账实行动态管理。发现有明显差异时,应提 出调整建议并报建设单位。 

# 7.5 进度监理

751 监理机构应在确保质量和安全 环保的基础上 严格控制进度计划 不 得任意压缩合同工期 

752 监理机构应要求施工单位在合同规定的期限内编制并提交进度计划, 进度计划应有文字说明 进度图表 资源配置和保证措施等 监理机构应审批施工 单位提交的总体进度计划和阶段性进度计划 签署审查意见后报建设单位批准 施工进度计划审查应包括下列基本内容: 

施工进度计划应符合施工合同中工期管理的约定,阶段性施工进度计划应 满足总进度目标控制的要求 

施工进度计划中主要工程项目应无遗漏 劳动力 材料 机械设备等应满足 施工需要 

施工进度计划应符合建设单位提供的资金、施工场地等条件。 

753 监理机构应定期检查工程实际进度,比较分析工程施工实际进度与计 划进度 预测实际工程进度对工程总工期的影响 并在监理月报中向建设单位 报告。 

754 总体工程进度严重滞后于计划进度时,监理机构应签发监理工作指令, 要求施工单位采取有效措施加快工程进度,同时向建设单位报告工期延误风险。 需要调整进度计划的,施工单位获得延期批复后,监理机构应要求施工单位根据延 期批复对工程进度计划进行调整,将调整后的工程进度计划报监理机构审查,并报 建设单位。 

755 由于施工单位原因造成工程进度延误,并可能影响合同工期时,监理机 构应及时向建设单位提交书面报告,并按合同有关规定处理。 

756 建设单位或施工单位提出工程进度较大调整时,应按施工合同或签订 的补充合同执行。 

# 8 机电工程监理

# 8.1 一般规定

811 公路机电工程包括公路监控系统 通信系统 含通信管道 收费系统 供配电系统 照明系统 隧道机电系统 

812 公路机电工程监理划分为下列四个阶段: 

施工准备阶段监理 监理合同签订之日至合同工程开工令确定的开工 之日。 

施工阶段监理:合同工程开工之日至合同工程完工初验申请受理之日。 

试运行阶段监理:合同工程完工初验申请受理之日至合同工程交工验收申 请受理之日,一般为 $3 \sim 6$ 个月。 

交工验收与缺陷责任期阶段监理 合同工程交工验收申请受理之日至缺陷 责任终止证书签发之日。 

813 监理企业应根据项目工程特点、工期要求组建现场机电监理机构。现 场机电监理机构应按合同要求配备机电工程监理人员和常规检测仪器、仪表等。 

814 公路机电工程监理除按本章规定开展监理工作外 还应执行本规范相 关规定。 

# 8.2 施工阶段监理

821 监理机构应按照下列规定审查进场的设备、材料: 

符合合同要求,具有产品检验合格证、质量检验单和出厂合格证。 

定制设备、材料的生产单位,必须具有相应的生产资质。 

进口设备、材料应提交商检部门的检验合格证书。 

进场的计算机平台软件应具有软件拷贝、说明书和最终的授权文件。 

822 经监理机构审查不合格的设备、材料、软件,必须限期清退出场,不得在 工程中使用。 

823 监理机构应审批施工单位提交的机电工程应用软件的需求分析、概要 设计、详细设计和测试大纲。应用软件必须经测试合格后,方可进行安装。 

824 监理机构应对施工单位安装完工并自检合格的单位、分部或分项工程设 备、线缆安装的质量、数量、位置、工艺进行工序验收。经验收合格的工程由监理机构 签发安装验收合格证书;未经安装验收或验收不合格的工程,不得进入调试工序。 

825 监理机构应按合同约定的系统功能、技术指标等内容审批施工单位提 交的系统测试大纲 

826 监理机构应检查施工单位使用的测试仪器仪表是否按规定进行了 校准。 

827 施工单位按测试大纲完成自测并提交自测报告后 监理机构应主持现 场系统检验测试,并对系统测试的各项指标进行评定。 

828 监理机构应审查施工单位提交的完工申请 具备完工条件的合同工程 应建议建设单位组织完工验收。 

829 监理机构应参加完工验收并签署意见。 

# 8.3 试运行阶段监理

831 监理机构应检查、督促施工单位按照完工验收提出的问题和意见进行 整改落实。 

832 监理机构应巡视系统的试运行情况,并做好巡视记录。重点检查试运 行人员的值班记录和系统工作情况。对发现的问题要求施工单位及时整改。 

833 监理机构应核查施工单位提供的专用工具、备品、备件的质量、数量是 否符合合同约定 建设单位 监理机构 施工单位 管养单位四方共同确认后签认 移交清单。 

834 监理机构应审查施工单位提交的交工申请,对具备交工验收条件的应 及时进行合同工程的质量检验评定。 

835 监理机构应审查施工单位提交的系统培训计划和培训教材 督促施工 单位做好系统维护与设备管理的培训工作 

# 9 交(竣)工验收与缺陷责任期监理

9.1 驻地监理工程师应按合同及有关规定,组织审查施工单位提交的合同工 程交工验收申请,签署意见后报总监理工程师审查、签认,并报建设单位。 

9.2 监理机构审查交工验收申请应包括下列内容: 

合同约定内容的完成情况。 

施工单位自检情况。 

施工资料完整性情况。 

工程数量核对情况。 

工程现场清理情况。 

9.3 监理机构应按相关规定及时汇总、整理监理资料,对工程的质量等级进 行评定,按有关规定编制监理工作报告,并提交建设单位。 

9.4 监理机构应参加建设单位组织的合同工程交工验收 接受建设单位对监 理抽检资料、监理工作报告及质量评定资料的检查,协助建设单位检查施工单位的 合同执行情况、检查工程实体、审查有关资料、核查工程完工数量。 

9.5 合同工程交工验收完成后,建设单位应组织及时签发交工证书,工程交 工证书应包括下列主要内容: 

交工验收时间。 

本合同段主要工程量。 

本合同段原合同价款 实际价款 

本合同段工期、实际工期。 

对工程质量、合同执行情况的评价、遗留问题、缺陷的处理意见及有关 决定。 

施工单位、监理企业、设计单位和建设单位的法定代表人或授权人签署 意见。 

9.6 监理机构应组织审核施工单位提交的合同工程交工付款申请单,提出建 设单位应支付给施工单位的价款,送建设单位审核并抄送施工单位。总监理工程 师向施工单位出具建设单位签认的交工付款证书。 

9.7 在合同工程的缺陷责任期内,监理工作主要包括: 

按合同约定对剩余工程进行监理。 

巡视检查已完工程,记录发生的工程缺陷,指令施工单位进行修复,并对工 程缺陷发生的原因、责任及修复费用进行调查、确认。 

督促施工单位按合同规定完成竣工资料。 

9.8 在合同工程缺陷责任期结束,收到施工单位向建设单位提交的终止缺陷 责任的申请后,驻地监理工程师应组织审查,签署意见后报总监理工程师审批、签 

认。对符合条件的,经建设单位同意,总监理工程师应在合同规定的时间内签发合 同工程缺陷责任终止证书,并向建设单位提交缺陷责任期监理工作总结。 

9.9 监理机构应组织核查施工单位提交的最终结清申请单,提出建设单位应 支付给施工单位的价款,送建设单位审核并抄送施工单位。总监理工程师向施工 单位出具建设单位签认的最终结清证书。 

9.10 监理机构应参加工程竣工验收工作,提交监理工作报告和工程监理资 料 并配合做好竣工验收检查工作 

# 10 合同其他事项管理

10.1 监理机构应按法律、法规、合同文件对工程分包进行审查,签署意见后 报建设单位审批。在监理过程中发现有转包、违法分包时,应要求施工单位改正并 报建设单位 

10.2 监理机构在监理过程中应按施工合同检查施工单位人员履约情况,重 点检查项目经理、技术负责人及其他关键岗位管理人员到岗履职情况,并检查进场 的施工机械设备是否符合施工合同约定,主要施工机械设备是否满足施工质量、安 全和进度等要求。 

10.3 监理机构应按监理合同约定的权限和下列规定审核工程变更: 

监理机构应组织专业监理工程师审查施工单位提出的工程变更申请,提出 审查意见 对涉及工程设计文件修改的工程变更 由建设单位转交设计单位修改工 程设计。必要时,应建议建设单位组织进行技术、经济论证。 

对建设单位提出的工程变更,按变更程序督促施工单位按变更组织实施。 

监理机构可向建设单位提出工程变更的建议。 

由工程变更引起的费用变化应按合同约定执行。 

10.4 监理机构应受理施工单位提交的符合施工合同约定的费用索赔意向通 知书 收集整理与索赔有关的资料 对其索赔原因 结果测算等进行审核 与建设单 位和施工单位协商一致后,编制费用索赔审核意见报告,报送建设单位。监理机构 同意施工单位费用索赔应同时满足下列条件 

施工单位在施工合同约定的期限内提出费用索赔申请。 

索赔事件因非施工单位原因造成,且符合施工合同约定。 

索赔事件给施工单位带来直接经济损失。 

因施工单位原因造成建设单位损失、建设单位提出索赔的,监理机构亦应 与建设单位和施工单位协商处理。 

10.5 监理机构签发工程停工令时,应明确工程暂停范围及工程暂停期间施 工单位应做的工作等 并报建设单位 

10.6 因施工单位原因引起的工程暂停需复工时 施工单位应向监理机构提 交复工申请,经审查暂停原因消除且具备复工条件时签发工程复工令,并报建设 单位。 

10.7 非施工单位原因引起的工程暂停 在暂停原因消除后具备复工条件时 监理机构应及时签发工程复工令 并报建设单位 

10.8 监理机构应对符合施工合同约定的延期意向或事件进行现场调查,并 应在施工单位提出工程延期申请后,对其延期原因、发展情况、拟采取的措施等进 行审核并报建设单位。监理机构同意工程延期应同时满足下列条件: 

施工单位在施工合同约定的期限内提出工程延期申请。 

非施工单位原因造成施工进度滞后。 

施工进度滞后的影响超出了施工合同约定的工期。 

10.9 当发生工期延误时,监理机构应及时调查、掌握情况和收集相关资料, 按施工合同约定提出处理意见 并报建设单位 

10.10 监理机构应按合同规定核定价格调整和计日工,并报建设单位。 

10.11 监理机构认为可能发生严重违约事件时 应及时提示建设单位和施工 单位。当违约事件发生时,监理机构应按规定进行调查分析、损失评估,提出处理 意见 并报建设单位 

10.12 监理机构在处理施工合同争议时应进行下列工作: 

了解合同争议情况,并调查和收集相关资料。 

及时与合同争议双方进行磋商。 

根据所掌握的情况、相关资料及有关规定,提出处理方案,由总监理工程师 进行协调,并报建设单位。 

监理机构在处理合同争议过程中 双方未达到施工合同约定的暂停履行合 同条件的,应要求合同双方继续履行合同。 

在施工合同争议仲裁或诉讼过程中,监理机构应按仲裁机构或法院要求提 供与争议有关的证据。 

10.13 施工合同解除时,监理机构应根据合同和相关规定,与建设单位及施 工单位协商确定施工单位应得款项或退还建设单位款项,签发工程结账证书。 

# 工地会议

# 11.1 工地会议的形式及纪要

1111 工地会议按召开时间 主题及参加人员的不同 分为第一次工地会议 工地例会、专题工地会议三种形式。 

1112 工地会议应由主持单位做好记录 整理成会议纪要 与会各方代表应 会签,主持单位签发,与会单位应签收。会议纪要应作为合同文件的一部分;会议 决定执行的有关事项,仍应按规定的监理程序办理。 

# 11.2 第一次工地会议

1121 会议应按下列要求组织: 

第一次工地会议应在工程正式开工前召开,由总监理工程师主持。 

总监理工程师办公室应事先将会议议程及有关事项通知建设单位、监理企 业、施工单位及其他有关单位并做好会议准备。 

建设单位 施工单位法定代表人或授权代表 参建各方主要技术管理人员 必须参加。 

邀请工程质量安全监督机构参加。 

1122 会议一般包括下列主要内容: 

建设单位、施工单位和监理企业分别介绍各自驻现场的组织机构、主要人 

员及其分工。 

建设单位宣布对总监理工程师的授权,总监理工程师宣布对驻地监理工程 师的授权 施工单位应提交对项目负责人 项目经理 的授权书 

建设单位介绍工程开工准备情况。 

监理机构介绍监理工作准备情况。 

施工单位介绍施工准备情况。 

总监理工程师介绍监理计划(或监理实施办法)的主要内容。 

研究确定工地例会相关事项。 

总监理工程师进行会议总结,明确是否具备开工条件及施工准备工作中存 在的主要问题、解决措施与要求。

解决需要协调的有关事项。

# 12 监理资料

# 12.1 一般规定

1211 监理资料应包括监理管理资料、监理工作记录资料和其他文件资料。 

1212 监理资料管理应遵循下列要求: 

监理机构应建立健全监理资料管理制度 

监理资料的管理应由总监理工程师或驻地监理工程师负责并指定专人具 体实施。 

监理机构应及时、齐全、真实、准确地收集、整理、编制、传递监理资料。 

监理资料应为原件 若为复印件 应加盖报送单位印章 并由经手人签字 注明日期。 

监理资料涉及的有关表格应采用本规范统一表式,签字盖章手续完备。 

1213 监理资料的归档应遵循下列要求: 

1 监理机构应及时整理、分类汇总监理资料,并应按规定组卷,形成监理 档案。 

监理档案应妥善存放和保管,按时移交建设单位。 

监理机构对未列入归档的监理资料也应分类整理,与工程直接相关的文件 资料,竣工后移交建设单位保管。
